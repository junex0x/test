# 🚀 JUNE

> Premium onboarding platform for **Shadowrocket setup** and **Locket Gold activation**

## Overview

**JUNE** is a modern web interface designed to help users:

- Set up **Shadowrocket** correctly
- Activate **Locket Gold** quickly
- Configure **DNS / iOS profile** smoothly
- Avoid common critical setup mistakes

## Features

- Clear step-by-step onboarding
- One-click configuration copy
- Embedded video tutorial
- DNS and iOS profile guidance
- Modern glassmorphism UI
- Fully optimized for mobile devices

## Tech Stack

- HTML5
- TailwindCSS
- Vanilla JavaScript
- Font Awesome
